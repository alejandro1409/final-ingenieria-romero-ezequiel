Consigna: crear proyecto local, contenerizarlo con Docker, versionarlo con Git y subirlo a un repositorio remoto. Incluí la URL del repositorio en la entrega. NO USAR EL MISMO APLICATIVO DEL TP.
Proyecto local: crear la carpeta final-ingenieria-apellido-nombre, armar una app mínima (HTML o servicio) con tu nombre y legajo, y agregar un README.

Docker: crear un Dockerfile, construir la imagen, exponer el puerto y comprobar que funcione. Documentar los comandos utilizados.

Git + Repositorio remoto: inicializar Git, realizar al menos tres commits, crear un repositorio remoto y subir el proyecto a la rama main.

Entrega: incluir la URL del repositorio y, opcionalmente, capturas mostrando el contenedor Docker en ejecución.
